# WebRTC Video Chat Application

A real-time multi-peer video conferencing application built using:

- Next.js
- TypeScript
- WebRTC
- Socket.io
- Docker

## Features

- Peer-to-peer video calls
- Multi-participant mesh topology (up to 4 users)
- Microphone mute/unmute
- Camera on/off
- Real-time chat
- Docker containerized deployment

## Run locally

```bash
npm install
npm run dev
